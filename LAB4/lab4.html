¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Java es un lenguaje de programacion que es utilizado con un proposito general ya sea para aplicaciones web o de escritorio. Con el se puede realizar cualquier tipo de programa, ademas es un lenguaje independiente de la plataforma lo que lo hace compatible con cualquier ordenador del mercado. En cambio JavaScript es un lenguaje de programacion especifico para el desarrollo web HTML, no es un lenguaje compilado. Es interpretado por el navegador y se ejecuta directamente.
